Lấy Key cho Telegram Bot

Hướng dẫn sử dụng:

  1. Nhập token bot Telegram của bạn (lấy từ @BotFather)
  2. Nhập ID Telegram của bạn (có thể lấy từ @userinfobot)
  3. Nhấn "Tạo Key" để nhận key kết nối
  4. Sử dụng key này để cấu hình bot vượt link

Lưu ý: Không chia sẻ token bot hoặc key này với người khác để tránh bị lạm dụng.
